# Convel Website Vision

## Objetivo del sitio
Crear una experiencia web corporativa premium para Convel que refleje:
- experiencia
- confianza
- arquitectura contemporánea
- capacidad técnica
- elegancia institucional

## Sensación visual
- Sobrio
- Editorial
- Arquitectónico
- Premium
- Moderno
- Espacioso
- Cinematográfico

## Evitar
- Apariencia de template WordPress
- Diseño genérico de inmobiliaria
- Exceso de animaciones
- Interfaces saturadas
- Sombras exageradas
- Gradientes innecesarios

## Referencias visuales
- Apple
- Aesop
- ArchDaily
- Stripe Press
- Constructoras premium modernas

## Prioridades
1. Performance
2. UX
3. Dirección visual
4. Responsive real
5. SEO
6. Accesibilidad

## Stack
- Astro
- Tailwind
- Motion sutil
- Component architecture
