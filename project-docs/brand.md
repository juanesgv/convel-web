# Brand Guidelines — Convel

## Brand Context

Convel S.A.S. es una empresa colombiana de diseño y construcción con más de 65 años de experiencia en proyectos de infraestructura, arquitectura, interiorismo y construcción de alto impacto.

La nueva experiencia digital debe transmitir:

* solidez
* confianza
* sofisticación
* experiencia
* arquitectura contemporánea
* precisión técnica
* elegancia corporativa

El sitio web NO debe sentirse como una constructora genérica ni como un template corporativo tradicional.

La dirección visual debe ser:

* sobria
* premium
* limpia
* arquitectónica
* editorial
* moderna
* espaciosa

---

# Design System Source

El proyecto cuenta con un sistema de diseño previamente generado con Claude Design basado en:

* manual de marca oficial
* recursos gráficos corporativos
* identidad visual existente de Convel

Este sistema de diseño debe ser tomado como:

* referencia principal de branding
* guía visual oficial
* base para componentes
* referencia tipográfica
* referencia cromática
* referencia de lenguaje visual

Sin embargo, el sistema NO debe ser interpretado de forma literal o rígida.

El objetivo es evolucionar la experiencia digital hacia una percepción más premium y contemporánea.

---

# Important Color Direction

Aunque el color principal de la marca es el naranja corporativo, existe una decisión estratégica importante para este proyecto:

## El naranja NO debe dominar visualmente el sitio web.

La nueva experiencia digital debe evitar una sobrecarga visual basada en naranja.

El naranja debe:

* existir dentro de la experiencia
* respetar la identidad corporativa
* funcionar como color de acento
* aparecer en momentos estratégicos
* reforzar branding y llamados de atención

Pero NO debe:

* convertirse en el color dominante del layout
* cubrir grandes superficies
* saturar la interfaz
* competir constantemente con el contenido
* hacer que el sitio se vea agresivo visualmente

---

# Preferred Color Usage

Priorizar:

* colores neutros
* tonos oscuros elegantes
* blancos cálidos
* grises arquitectónicos
* colores secundarios del sistema de diseño
* contrastes sofisticados y sobrios

El naranja debe reservarse principalmente para:

* detalles interactivos
* hover states
* indicadores
* pequeños acentos visuales
* CTAs estratégicos
* elementos de branding específicos

# Technical Direction

El sitio será desarrollado usando:

* Astro
* Arquitectura basada en componentes
* Performance-first approach
* Responsive design real
* Accesibilidad AA
* SEO optimizado

La experiencia debe priorizar:

1. Performance
2. Claridad visual
3. Elegancia
4. Escalabilidad
5. Mantenibilidad

---

# AI Collaboration Rules

Antes de generar código o propuestas visuales:

* analizar contexto
* justificar decisiones
* evitar clichés de constructoras
* evitar patrones genéricos SaaS
* evitar interfaces saturadas
* priorizar composición y jerarquía visual

La IA debe actuar como:

* colaborador senior
* director frontend asistente
* UX/UI partner
* system thinker

No como generador automático de layouts.
